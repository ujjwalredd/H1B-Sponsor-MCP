# Support

Use GitHub issues for public project support:

https://github.com/ujjwalredd/H1B-Sponsor-MCP/issues

Good public issue topics include:

- Installation problems.
- MCP client configuration.
- Incorrect or confusing documentation.
- Reproducible tool behavior bugs.
- Dataset interpretation questions that do not include private information.

Do not post secrets, private immigration records, personal documents, or
vulnerability details in public issues.

For vulnerabilities, follow [SECURITY.md](SECURITY.md):

https://github.com/ujjwalredd/H1B-Sponsor-MCP/security/advisories/new

For conduct concerns, follow [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
